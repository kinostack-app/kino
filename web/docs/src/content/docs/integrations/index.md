Kino works on its own, but plugs into the services you already use. All
four integrations live under **Settings → Integrations** and are
optional — kino acquires, organises, and plays media without any of
them.

## Pages in this section

| Page | What it covers |
|---|---|
| [Trakt](./trakt) | Two-way sync of watched / ratings / collection / watchlist + live scrobbling during playback |
| [Lists](./lists) | Follow curated lists from MDBList, TMDB, and Trakt |
| [OpenSubtitles](./opensubtitles) | Auto-fetch subtitles next to each imported file |
| [Notifications (webhooks)](./notifications) | Fire JSON or plain-text webhooks to Discord, Slack, ntfy, Telegram, or anything that accepts HTTP |

If you're setting kino up for the first time, finish the
[Quickstart](../getting-started) and
[indexer setup](../setup/indexers) before turning to integrations
— they're enhancements, not prerequisites.
