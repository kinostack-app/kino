These pages document what Kino actually does once it's installed and
configured. They're organised by feature, not by settings tab — pick
whichever one matches what you're trying to do.

## In this section

| Page | What it covers |
|---|---|
| [Library and automation](./library) | Following movies and shows, monitoring policies, library navigation |
| [Streaming and transcode](./streaming) | Direct play, hardware transcode, subtitles, scrubbing previews |
| [Casting](./casting) | Sending playback to a Chromecast from any browser |
| [Intro and credits skipping](./intro-skipping) | Automatic detection of TV intros and end credits |
| [Backup and restore](./backup-restore) | Automatic backups, retention, restoring after disk loss |
| [Home customisation](./home-customisation) | Reordering Home rows, toggling the hero, pinned lists |

If you're still installing, start with the
[Quickstart](../getting-started) for your platform. If you're stuck
on a specific behaviour, the
[Troubleshooting FAQ](../troubleshooting/faq) is shorter and more
direct than these pages.
